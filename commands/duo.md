# Duo — Collaborative Pair Programming

Start a collaborative coding session where you write the interesting code and AI handles the boilerplate.

## Usage
/duo:start [project description]

## Workflow
1. **Design** — Discuss and agree on approach
2. **Plan** — Break down tasks, assign human vs AI
3. **Execute** — Code in parallel
4. **Review** — Cross-review each other's code  
5. **Integrate** — Merge, test, commit
